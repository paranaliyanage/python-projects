This is the implementation of google code jam question[0], Triangle Trilemma. 
This code may contain errors, but it is working fine for the small data set.

[0] http://code.google.com/codejam/contest/dashboard?c=agdjb2RlamFtcg4LEghjb250ZXN0cxhRDA

Usage
-----
You can input small or large input files given in the input folder. If you are in the
root folder you can enter,

    $python src/main.py small

use large if you want to run the large input data file. By default it will take the small
input data file.
